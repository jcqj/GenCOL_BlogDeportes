Quiénes Somos

Somos un grupo de estudiantes que creó este blog de deportes como parte de un proyecto académico.

Nuestro equipo

Nuestro equipo está conformado por estudiantes que trabajan juntos para desarrollar un sitio web funcional y fácil de usar.

Nuestra misión

Nuestra misión es compartir información sobre deportes y aprender desarrollo web utilizando HTML, CSS y GitHub.

¿Por qué creamos este blog?

Creamos este blog para compartir noticias, curiosidades y contenido relacionado con diferentes deportes como fútbol, ciclismo y demas.
